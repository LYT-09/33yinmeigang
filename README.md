# 校园二手交易平台
## 项目介绍
本项目是基于Java Web技术栈开发的校园专属二手交易平台，专为在校师生打造，提供商品发布、分类浏览、订单管理、用户认证等核心功能，解决校园闲置物品流转信息分散、交易不规范的问题，实现校内资源高效循环利用。

## 使用方法
本项目用于搭建安全、便捷的校园二手交易场景，支持师生发布闲置物品、检索目标商品、发起交易、管理订单，覆盖学习用品、生活用品、数码产品等全品类，助力绿色低碳校园建设。

1.  注册并完成校园身份认证，登录平台后即可发布闲置商品或浏览全平台二手物品。
2.  通过分类筛选或关键词搜索找到目标商品，发起交易并完成订单流程，在个人中心管理订单与商品。
